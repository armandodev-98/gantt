📋 Introducción

Gestor Profesional de Proyectos te permite organizar y dar seguimiento a tus proyectos mediante etapas, actividades y cronogramas visuales. Ideal para equipos de trabajo que necesitan controlar plazos, responsables y progreso.

🏢 Gestión de Proyectos Básico
  • Crear proyecto: Haz clic en "Nuevo" y asigna un nombre y responsable.
  • Editar proyecto: Usa el botón "Datos" para modificar responsable y fechas del proyecto.
  • Cambiar entre proyectos: Selecciona la pestaña del proyecto que deseas visualizar.
  • Exportar/Importar: Puedes guardar un proyecto individual o hacer backup completo de todos los proyectos.
📌 Gestión de Etapas Básico
  • Crear etapa: Haz clic en "Nueva Etapa" y define nombre y fechas.
  • Validación: Las fechas de etapa deben estar dentro del rango del proyecto.
  • Colapsar/Expandir: Haz clic en el encabezado de la etapa para ocultar o mostrar su contenido.
  • Editar/Eliminar: Usa los botones en la parte derecha de cada etapa.
✅ Gestión de Actividades Intermedio
  • Crear actividad: Dentro de una etapa, haz clic en "Nueva Actividad".
  • Prioridad: Asigna prioridad Alta, Media o Baja (visualmente codificada por colores).
  • Fechas: Deben estar dentro del rango de la etapa y del proyecto.
  • Progreso: Actualiza el porcentaje de avance usando el deslizador.
  • Dependencias: Una actividad puede depender de otra, afectando su orden.
  • Comentarios: Cada actividad puede tener comentarios. Haz clic en el botón 💬 para gestionarlos.
📝 Notas por Etapa Básico
  • Agregar notas: Usa el botón "Nueva Nota" dentro de cada etapa.
  • Las notas son informativas: No afectan el progreso ni las fechas, ideales para recordatorios o ideas.
  • Eliminar notas: Cada nota tiene un botón para eliminarla.
📊 Reportes y Exportación Intermedio
  • Reporte PDF: Genera un informe profesional con todas las etapas, actividades, prioridades, progreso y comentarios.
  • Búsqueda global: Encuentra actividades por nombre, responsable o comentarios. Genera reporte de resultados.
  • Exportar proyecto: Guarda un proyecto específico como archivo JSON.
  • Backup completo: Exporta todos los proyectos y configuraciones como respaldo.
📈 Diagrama de Gantt Avanzado
  • Visualización: Muestra todas las actividades en una línea de tiempo.
  • Personalización: Ajusta altura de barras, ancho de columnas, colores y vista (Día/Semana/Mes).
  • Rango manual: Puedes definir un rango de fechas específico para el diagrama.
  • Interactivo: Haz clic en cualquier barra para editar la actividad.
¿Cómo establezco fechas para el proyecto?
Haz clic en el botón "Datos" en la sección de proyectos. Allí podrás definir el responsable y las fechas de inicio y fin del proyecto. Estas fechas servirán como límites para las etapas.
¿Qué pasa si una actividad tiene fecha fuera del rango?
El sistema mostrará una advertencia visual (fondo rojo) y no permitirá guardar la actividad hasta que las fechas estén dentro del rango permitido.
¿Cómo funcionan las dependencias entre actividades?
Al crear o editar una actividad, encontrarás una sección "Dependencias". Puedes seleccionar de qué otras actividades depende. Esto ayuda a visualizar el orden lógico en el diagrama de Gantt.
¿Puedo tener múltiples proyectos?
Sí, puedes crear todos los proyectos que necesites. Usa las pestañas superiores para cambiar entre ellos. El proyecto actual se muestra resaltado.
¿Cómo recupero mi información si cierro el navegador?
Todos los datos se guardan automáticamente en el almacenamiento local de tu navegador (localStorage). Al volver a abrir la página, tus proyectos se cargarán automáticamente. Para mayor seguridad, usa la opción "Backup" para exportar tus datos.
¿Puedo editar una actividad después de crearla?
Sí, cada actividad tiene un botón de edición (✏️) que te permite modificar todos sus campos: nombre, responsable, prioridad, fechas, progreso, color y dependencias.
¿Cómo genero un reporte profesional?
Haz clic en el botón "PDF" en la sección superior. Se abrirá una nueva ventana con el reporte listo para imprimir o guardar como PDF (Ctrl+P).
¿Qué significan los colores en las actividades?
Rojo = Prioridad Alta / Atrasado, Amarillo = Prioridad Media / Próximo a vencer, Verde = Prioridad Baja / Progreso normal, Gris = Actividad completada.
